There are $$$n$$$ bullets in the magazine, the $$$i$$$-th of which deals $$$a_i$$$ damage. The enemy starts with $$$h$$$ health and dies when its health becomes $$$\le 0$$$.</p><p>The gun shoots one bullet per second. After firing all $$$n$$$ bullets, it must reload, which takes $$$k$$$ seconds. Reloading always restores the <span class="tex-font-style-bf">same sequence</span> of bullets $$$[a_1, a_2, \ldots, a_n]$$$. You cannot reload early; you must empty the magazine first. At the start, the magazine is already full.</p><p>Before the fight begins, you may perform <span class="tex-font-style-bf">at most one</span> swap: pick any indices $$$1 \le i  \lt  j \le n$$$ and exchange $$$a_i$$$ with $$$a_j$$$.</p><p>Your task is to find the minimum number of seconds needed to kill the enemy, taking into account this optional single swap.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. 10^9$$$).</p><p>It is guaranteed that the sum of $$$n$$$ does not exceed $$$2 \cdot 10^5$$$ over all test cases.</p></div><div class="output-specification"><div
