Of course, Bob wants to know if someone did alter the string!</p><p>You are given a string $$$X$$$ of length $$$n$$$, which consists of '<span class="tex-font-style-tt">a</span>', '<span class="tex-font-style-tt">b</span>', and '<span class="tex-font-style-tt">?</span>'.</p><p>Please determine if there exists a string $$$A$$$ which satisfies the following conditions:</p><ul> <li> $$$|A|=n$$$; </li><li> $$$A_i$$$ is either '<span class="tex-font-style-tt">a</span>' or '<span class="tex-font-style-tt">b</span>' for all $$$1 \le i \le n$$$; </li><li> $$$A_i=X_i$$$ for all $$$1 \le i \le n$$$ such that $$$X_i$$$ is not '<span class="tex-font-style-tt">?</span>'; </li><li> The string $$$A$$$ can be generated from the algorithm described above. </li></ul></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases.
