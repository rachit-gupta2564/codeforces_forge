<h3><a href="https://codeforces.com/contest/2205/problem/F" target="_blank" rel="noopener noreferrer">Simons and Reconstructing His Roads</a></h3>

<div class="header"><div class="title">F. Simons and Reconstructing His Roads</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><div class="epigraph"><div class="epigraph-text"><span class="tex-font-style-it">Dropping all the time, but the exit I can find.</span></div><div class="epigraph-source">— SHUN, <span class="tex-font-style-it"><a href="https://open.spotify.com/track/2akBCXjdlUgpyZ4cr1t3B9">DYING FOR YOU</a></span></div></div><p>There are $$$n \times m$$$ crossroads in the city, named crossroad $$$(1, 1), (1, 2),\ldots, (n, m)$$$. The first number represents the row, and the second represents the column.</p><p>There exists and only exists streets between $$$(i,j)$$$ and $$$(i+1,j)$$$ or between $$$(i,j)$$$ and $$$(i,j+1)$$$. The street between $$$(i, j)$$$ and $$$(i + 1, j)$$$ has a weight of $$$w_{i,j}$$$, and the street between $$$(i, j)$$$ and $$$(i, j + 1)$$$ has a weight of $$$v_{i,j}$$$.</p><p>Simons wants some streets reconstructed. Due to some accidents, some of the streets <span class="tex-font-style-bf">cannot</span> be reconstructed, while others are optional to be reconstructed.</p><p>For each crossroad, if the number of reconstructed streets adjacent to it is <span class="tex-font-style-bf">even</span>, Simons calls the crossroad <span class="tex-font-style-it">elegant</span>. If all the crossroads are <span class="tex-font-style-it">elegant</span>, Simons calls the reconstruction <span class="tex-font-style-it">nice</span>. </p><p>The <span class="tex-font-style-it">beauty</span> of a reconstruction is calculated as follows:</p><ul> <li> At first, the <span class="tex-font-style-it">beauty</span> is $$$0$$$. </li><li> For each <span class="tex-font-style-bf">row</span> $$$i$$$ from $$$1$$$ to $$$n - 1$$$, let the columns of the streets reconstructed be $$$c_1 \lt c_2 \lt c_3 \lt  \cdots$$$, add $$$w_{i, c_1} - w_{i, c_2} + w_{i, c_3} - w_{i, c_4}+\cdots$$$ to the <span class="tex-font-style-it">beauty</span>. </li><li> Similarly, for each <span class="tex-font-style-bf">column</span> $$$j$$$ from $$$1$$$ to $$$m - 1$$$, let the rows of the streets reconstructed be $$$r_1 \lt r_2 \lt r_3 \lt  \cdots$$$, add $$$v_{r_1, j} - v_{r_2, j} + v_{r_3, j} - v_{r_4, j}+\cdots$$$ to the <span class="tex-font-style-it">beauty</span>. </li></ul><p>In other words, if a street is an odd-indexed one reconstructed in its row or column, add its weight to the <span class="tex-font-style-it">beauty</span>; else subtract it from the <span class="tex-font-style-it">beauty</span>.</p><p>For example, consider the streets and crossroads below without any streets that cannot be reconstructed:</p><center> <img class="tex-graphics" src="https://espresso.codeforces.com/5826d6aa9c0b7895bcf04a1f291b3c18c586aa08.png" style="zoom: 50.0%;max-width: 100.0%;max-height: 100.0%;"> </center><p>We can have a <span class="tex-font-style-it">nice</span> reconstruction as follows:</p><center> <img class="tex-graphics" src="https://espresso.codeforces.com/0c09dd7c7f03acecc55b71026b667499ee254912.png" style="zoom: 50.0%;max-width: 100.0%;max-height: 100.0%;"> </center><p>The <span class="tex-font-style-it">beauty</span> of the reconstruction is $$$3+4+2-(-9)-(-2)+9+1-(-1)-(-3)-(-4)=38$$$.</p><p>Help Simons find the maximum <span class="tex-font-style-it">beauty</span> among all the <span class="tex-font-style-it">nice</span> reconstructions.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 5\cdot 10^4$$$). The description of the test cases follows. </p><p>The first line contains two integers $$$n$$$ and $$$m$$$ ($$$2\le n,m\le 2\cdot 10^5$$$; $$$n\cdot m\le 4\cdot 10^5$$$) — the number of rows and columns.</p><p>In the next $$$n-1$$$ lines, each line contains $$$m$$$ integers $$$w_{i,j}$$$ ($$$-10^9\le w_{i,j}\le 10^9$$$) — the weights of the street between $$$(i,j)$$$ and $$$(i+1,j)$$$.</p><p>In the next $$$n$$$ lines, each line contains $$$m-1$$$ integers $$$v_{i,j}$$$ ($$$-10^9\le v_{i,j}\le 10^9$$$) — the weights of the street between $$$(i,j)$$$ and $$$(i,j+1)$$$.</p><p>In the next $$$n-1$$$ lines, each line contains $$$m$$$ characters $$$p_{i,j}$$$ ($$$p_{i,j}\in\{\texttt{0},\texttt{1}\}$$$) — if $$$p_{i,j}=\texttt{0}$$$, the street between $$$(i,j)$$$ and $$$(i+1,j)$$$ cannot be reconstructed, and vice versa.</p><p>In the next $$$n$$$ lines, each line contains $$$m-1$$$ characters $$$q_{i,j}$$$ ($$$q_{i,j}\in\{\texttt{0},\texttt{1}\}$$$) — if $$$q_{i,j}=\texttt{0}$$$, the street between $$$(i,j)$$$ and $$$(i,j+1)$$$ cannot be reconstructed, and vice versa.</p><p>It is guaranteed that the sum of $$$n\cdot m$$$ over all test cases does not exceed $$$4\cdot 10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer — the maximum <span class="tex-font-style-it">beauty</span> among all the <span class="tex-font-style-it">nice</span> reconstructions.</p></div><div class="sample-tests"><div
